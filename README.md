# Fawad Farooq Ashraf Portfolio

Static academic portfolio for GitHub Pages:

> https://fawadkhanf3.github.io/

## Editing Content

Most portfolio content lives in `assets/js/site-data.js`.

- Add education, experience, publications, projects, certificates, interests, and skills by editing the arrays in that file.
- Keep images and PDFs under `assets/` and reference them with relative paths, for example `assets/img/project/example.png`.
- The page layout is rendered by `assets/js/site.js`.
- The visual theme is controlled by `assets/css/portfolio.css`.

## Local Preview

From the repository root:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
